
My solutions (both Python, Rust and Kotlin) for [Advent of Code 2021](https://adventofcode.com/2021/about).

### Running

Rust:
```
$ cd rust && ./run.sh
```

Python:
```
$ cd python && ./run.sh
```

Kotlin:
```
$ cd kotlin && ./run.sh
```

There are also corresponding Jupyter notebooks for both Python, Rust and Kotlin.
